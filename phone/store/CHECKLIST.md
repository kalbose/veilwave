# Чеклист RuStore — RuSamaraWave

1. Аккаунт разработчика RuStore (Владелец компании).
2. Создать приложение: имя **RuSamaraWave**, пакет `ru.samara.wave`.
3. Загрузить AAB или APK: `RuSamaraWave-release.aab` / `.apk`.
4. Иконка 512×512: `store/icon-512.png`.
5. Скриншоты ≥2 (телефон).
6. Тексты из `LISTING.md`.
7. Политика конфиденциальности — URL на `privacy.html` (или текст из него).
8. Декларация разрешений + «данные не собираются».
9. Отправить на модерацию.
10. После публикации — проверить установку из RuStore.
