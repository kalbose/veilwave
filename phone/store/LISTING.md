# RuSamaraWave — материалы для RuStore

## Краткое описание (до 80 символов)

Офлайн-заметки голосом в защищённом формате .rswk — всё только на телефоне.

## Полное описание

RuSamaraWave — бесплатное приложение для личных голосовых заметок.

Что умеет:
• записать голос с микрофона;
• сохранить заметку в формате .rswk с кодовым словом;
• открыть .rswk тем же ключом и прослушать / сохранить WAV.

Особенности:
• работает офлайн, без аккаунта и без сервера;
• шифрование на устройстве (Argon2id + XChaCha20-Poly1305);
• файлы сохраняются в папку «Загрузки/RuSamaraWave».

Важно: надёжность зависит от сложности кодового слова. Не используйте простые пароли и не храните ключ в том же чате, куда отправляете файл.

## Разрешения (обоснование для консоли)

| Разрешение | Зачем |
|---|---|
| RECORD_AUDIO | Запись голосовой заметки по нажатию пользователя |
| INTERNET | Техническое требование WebView Capacitor; пользовательские данные не отправляются |
| WRITE_EXTERNAL_STORAGE (API ≤28) | Сохранение файлов на старых Android |

## Чеклист перед загрузкой

- [ ] Собрать signed AAB/APK (`npm run build:rsw && npx cap sync android && npm run apk:release`)
- [ ] Проверить запись → .rswk → открытие на реальном телефоне
- [ ] Загрузить icon-512.png
- [ ] Добавить 2–4 скриншота (сделать на устройстве)
- [ ] Указать ссылку на политику конфиденциальности (можно разместить privacy.html на странице VK / GitHub Pages)
- [ ] Заполнить блок «Безопасность данных»: нет сбора ПДн
- [ ] Подпись: использовать keystore из `android/app/keystore/` (пароли в keystore.properties — не публиковать)
